# Cloud Roadmap (2026)

If I had to start cloud today, I'd go like this:

### 1. Fundamentals
What is cloud? IaaS vs PaaS vs SaaS, Regions, AZs, pricing basics

### 2. Networking
VPC, Subnets, Routing, NAT, DNS (Security Groups vs NACLs is a must)

### 3. Compute
EC2 / VMs, Autoscaling, Load Balancers — how apps actually run in cloud

### 4. Storage
S3 / Blob storage, EBS, Lifecycle policies — when to use what

### 5. Databases
RDS, NoSQL (DynamoDB / Firestore), Backups, replication, scaling

### 6. IAM & Security
Users, Roles, Policies — least privilege mindset

### 7. Cloud CLI + SDKs
AWS CLI / gcloud — automate infra interactions

### 8. Infrastructure as Code
Terraform / CloudFormation — no manual clicking

### 9. Containers in Cloud
ECS / EKS / GKE — deploy containerized apps

### 10. Monitoring & Logging
CloudWatch / Stackdriver — alerts, metrics, logs

### 11. Cost Optimization
Billing dashboards, budgeting, rightsizing — cloud gets expensive fast

### 12. High Availability & Disaster Recovery
Multi-AZ, failover, backup strategies, RTO vs RPO

### 13. Real Projects
Build and deploy real apps on cloud to cement everything learned
